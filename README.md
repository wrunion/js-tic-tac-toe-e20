# Rainbow Tic Tac Toe

#### _A responsive tic tac toe game, built with Flexbox and CSS Grid. 2.16.2020_

* View the app [here](https://wrunion.github.io/rainbow-tic-tac-toe/)
* Grab a friend.
* Play the game!
